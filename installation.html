Frostbite

A frontend for Dragonrealms MUD

1.2 Installation

Application software

No installation is required. The binary package is compressed into an archive and has to be extracted once downloaded.


Download:
  • - Windows 8 zip archive - click here to download.
  • - LInux Ubuntu tar.gz archive - click here to download.
  • - Mac OS X Snow leopard dmg archive - click here to download.

Install instructions:
  1. 1. Download archive.
  2. 2. Extract the contents of the archive into your desired location.
  3. 3. Run FrostBite(.exe, .app).

For older versions of Mac OS X (Snow Leopard) shared memory parameters need to be readjusted to allow data exchange between scripting module and frontend (http://support.apple.com/kb/HT4022).
To determine system settings run following command in terminal:
  • sysctl kern.sysv.shmseg
The value of "kern.sysv.shmseg" needs to be at least 40.
  • Example values for Mountain Lion:
  • kern.sysv.shmmax=134217728
  • kern.sysv.shmmin=1
  • kern.sysv.shmmni=256
  • kern.sysv.shmseg=64
  • kern.sysv.shmall=3276
To change shared memory settings create a file called "/etc/sysctl.conf" and copy the values for Mountain Lion.

Ruby runtime

Ruby runtime installation is necessary for running scripts in the Frostbite frontend.

Frostbite scripting module has been verified to run with Ruby runtime version 1.9.3 and it uses Ruby DL library which was first introduced into Ruby since version 1.9. Accordingly, versions lower than 1.9 will not be suited for running the Frostbite scripting core API.


Windows

The recommended version of Ruby runtime 1.9.3 installation package for Windows can be downloaded from http://cdn.rubyinstaller.org/archives/1.9.3-p327/rubyinstaller-1.9.3-p327.exe


Install instructions:
  1. 1. Download Ruby installer.
  2. 2. Run Ruby installer.
  3. It is important to include Ruby executables in PATH.
  4. Example:
  6. 3. Verify installation.

Ruby installation can be verified by running the "Ruby --version" command in Windows command prompt. 

        C:\>ruby --version
        ruby 1.9.3p194 (2012-04-20) [i386-mingw32]

Linux Ubuntu

  1. 1. sudo apt-add-repository ppa:brightbox/ruby-ng
  2. 2. sudo apt-get update
  3. 3. sudo apt-get install ruby1.9.3
  4. 4. ruby -v

Mac OS X

Mac OS X already comes with a Ruby installation prepackaged. For OS X it is important to upgrade Ruby if the Ruby version is older than 1.9.3. To find out which version of ruby is installed run the following command in terminal: 

        ruby -version
Updating Ruby using RVM (http://rvm.io/rvm/upgrading):
  1. 1. \curl -sSL https://get.rvm.io | bash
  2. 2. rvm get stable
  3. 3. rvm reload
  4. 4. rvm list
  5. 5. rvm install 1.9.3

If it was necessary to upgrade Ruby then execute the following steps:

Register new ruby installation in the Frostbite frontend.
  1. 1. which ruby
  2. for example => /Users/<*user*>/.rvm/rubies/ruby-1.9.3-p545/bin/ruby

Navigate to the location where the application was extracted during installation.
  1. 1. cd FrostBite.app/Contents/MacOS
  2. 2. open/create file called client.ini
  3. 3. copy ruby path to client.ini:

            [Ruby]
            path=/Users/<*user*>/.rvm/rubies/ruby-1.9.3-p545/bin/ruby
