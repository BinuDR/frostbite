Frostbite

A frontend for Dragonrealms MUD

1.1 Application software

The Frostbite application is built into a binary software package which includes the application executable and all the library files required to run the application. In addition to the application software package, the Frostbite application requires the Ruby runtime (is not included in the software package).

The ruby runtime installation is required to run scripts in the Frostbite frontend however the client is still available to use without any support from Ruby. The installation of ruby runtime is described in more detail in the installation page.

The application software can be downloaded from the installation page.


The mechanics in scripting module implementation restrict from running more than one frontend per application copy.


The application structure

The Frostbite client software comes in a pre-defined structure for containing the necessary utility files required by different functions of the application.


  • /..
  • docs
    • User guide documents are placed in this folder.
  • notepad
    • Notepad2 is used to open scripts for editing from the scripts menu.
  • profiles
    • Contains client settings profiles.
  • scripts
    • The designated folder for the application scripts.
  • sounds
    • The designated folder for sound files used by highlights and alerts.

  • data.dll - contains shared memory of game data which is used by the scripts.
  • FrostBite.exe - application executable.
  • client.ini - settings file for general client settings.
  • highlights.ini - settings file for color highlights.
  • macros.ini - settings file for macros.
  • general.ini - general purpose settings file.
