About

Frostbite is a desktop MUD frontend for Windows and it is specifically designed for Dragonrealms. It is free to use and is created based on open source principles -- allowing any personal modifications or additions to the current software.

The application software is developed in C++ using Qt framework which enables to extend the application across multiple platforms to Linux and Mac OS in future releases.

The Frostbite frontend includes full-featured user interface; text highlighting; macros and sequences; and scripting support with Ruby scripting language -- all of which will be covered in more detail in this document.
