Frostbite

A MUD client for Dragonrealms

4 Development

The Frostbite frontend application has been programmed in c/c++ using Qt framework version 4.8. The development project is set up in Qt Creator IDE. All the Qt tools are free to use for open source projects and can be downloaded from the Qt website.


Github

The application source code is hosted in the Github repository - https://github.com/matoom/frostbite. It is possible to contribute to the Frostbite project in Github by forking the repository as it is described in the following guide - https://help.github.com/articles/fork-a-repo


Setting up the project

The following steps are necessary to set up the project in Qt Creator IDE in Windows:

  • Open Frostbite.pro in Qt Creator;
  • Load the environment settings -> No -> Select Desktop project;
  • In environment.h -> change project path for offline mock file - #define MOCK_DATA_PATH "C:/Projects/FrostBite/support/mock.xml";
  • in connectionmanager.cpp -> set debug = true; to run client in offline development mode;
  • Build for debug target;
  • In projects tab -> add a custom executable -> find exe file in debug build folder;
  • Run project in debug mode.

High level architecture

The entry point for the Frostbite application is main.cpp which runs the mainwindow.cpp object.

The MainWindow class creates instances for the following files:

  • windowmanager.cpp
  • toolbarmanager.cpp
  • timerbar.cpp
  • commandline.cpp
  • connectionmanager.cpp
  • scriptservice.cpp
  • menuhandler.cpp
