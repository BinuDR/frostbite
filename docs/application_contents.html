Frostbite

A MUD client for Dragonrealms

1.1 Application contents

As a whole, the application is comprised of the application software and Ruby runtime (which is not included in the installation package and has to be downloaded separately).

By itself the application software is capable of running without any support from Ruby but it is necessary for running scripts from the client. The installation of ruby runtime is described in more detail in the installation page.


The application structure

The Frostbite client software comes in a pre-defined structure for containing the necessary utility files required by different functions of the application.


  • /..
  • docs
    • User guide documents are placed in this folder.
  • notepad2
    • Notepad2 is used to open scripts for editing from the scripts menu.
  • scripts
    • The designated folder for the application scripts.
  • sounds
    • The designated folder for sound files used by highlights and alerts.

  • data.dll - contains shared memory of game data which is used by the scripts.
  • FrostBite.exe - application executable.

Due to mechanics in scripting module implementation only one instance of the game application can be run per copy.
